# 13. ↩️ 撤销工具

import useBaseUrl from '@docusaurus/useBaseUrl';

撤销工具用于撤销上一步操作，恢复到之前的状态。

**使用方法：**
1. 点击工具栏中的“撤销工具”图标（↩️）。
2. 系统会自动撤销最近的一次操作。

<video controls width="480px" height="270px" src={useBaseUrl('/video/撤销工具.mp4')} />

**应用场景：**
- 误操作时快速恢复。
- 多步编辑时灵活调整。
