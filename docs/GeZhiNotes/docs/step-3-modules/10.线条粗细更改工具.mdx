# 10. ➖ 线条粗细更改工具

import useBaseUrl from '@docusaurus/useBaseUrl';

线条粗细更改工具用于调整绘图和连接线的粗细。

**使用方法：**
1. 在工具栏的 Size 输入框中输入所需的线条粗细数值。
2. 后续绘图和连接线将采用该粗细。

<video controls width="480px" height="270px" src={useBaseUrl('/video/线条粗细更改工具.mp4')} />

**应用场景：**
- 强调重要节点或关系。
- 美化画布视觉层次。
