# 9. 🎨 颜色选择和颜色自定义工具

import useBaseUrl from '@docusaurus/useBaseUrl';

颜色选择和自定义工具用于为绘图和节点连接选择颜色。

**使用方法：**
1. 在工具栏选择三种常用颜色之一，或点击自定义按钮手动设定任意颜色。
2. 颜色设定好后，后续绘图和连接线都将使用该颜色。

<video controls width="480px" height="270px" src={useBaseUrl('/video/颜色选择和颜色自定义工具.mp4')} />

**应用场景：**
- 区分不同类型节点或关系。
- 丰富画布视觉效果。
