# 4. 🖼️ 图片工具

import useBaseUrl from '@docusaurus/useBaseUrl';

图片工具用于在画布上插入本地图片，丰富内容表达。

**使用方法：**
1. 点击工具栏中的“图片工具”图标（🖼️）。
2. 在画布上点击需要插入图片的位置。
3. 选择本地图片文件，图片将自动添加到画布。
4. 可拖动、缩放图片，调整其大小和位置。

<video controls width="480px" height="270px" src={useBaseUrl('/video/图片工具.mp4')} />

**应用场景：**
- 添加插图、截图、照片。
- 辅助说明、视觉化表达。
