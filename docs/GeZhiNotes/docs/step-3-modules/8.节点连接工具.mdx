# 8. 🪢 节点连接工具

import useBaseUrl from '@docusaurus/useBaseUrl';

节点连接工具用于在画布上连接两个节点，建立它们之间的关系。

**使用方法：**
1. 点击工具栏中的“节点连接工具”图标（🪢）。
2. 依次点击两个需要连接的节点。
3. 系统会自动在两个节点之间绘制连接线。

<video controls width="480px" height="270px" src={useBaseUrl('/video/节点连接工具.mp4')} />

**应用场景：**
- 构建思维导图、知识网络。
- 表达节点间的层级、依赖或关联。
