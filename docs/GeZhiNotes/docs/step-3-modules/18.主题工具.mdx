# 18. 🌗 主题工具

import useBaseUrl from '@docusaurus/useBaseUrl';

主题工具用于切换画布的显示主题（如深色/浅色模式）。

**使用方法：**
1. 点击工具栏中的“主题工具”图标（🌗）。
2. 在弹出的主题选择菜单中选择所需主题。
3. 画布界面将自动切换到对应主题风格。

<video controls width="480px" height="270px" src={useBaseUrl('/video/主题工具.mp4')} />

**应用场景：**
- 夜间/白天切换，保护视力。
- 个性化界面风格。
